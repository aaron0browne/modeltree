#### Install

```bash
pip install modeltree
```

#### Setup

_This is technically optional, but recommended to get access to the [management commands]({% post_url 2013-07-20-commands %})._

Add `modeltree` to Django's `INSTALLED_APPS` setting.

```python
INSTALLED_APPS = (
    ...
    'modeltree',
)
```
